## 详细接口清单（选取关键接口）

### 1. 认证与用户（AuthUserController）
- POST /api/auth/login
  - 请求体：`LoginDTO` (username, password)
  - 返回：JWT token（字符串）
  - 说明：普通登录入口。
  - 源码：[src/main/java/com/example/onlyone/Controller/User/AuthUserController.java#L1]

- POST /api/auth/register
  - 请求体：`RegisterDTO`
  - 返回：Result
  - 说明：用户注册。

- POST /api/auth/QQ/sendCode
  - 请求体：{"email": "..."}
  - 返回：是否发送成功（Boolean）

- POST /api/auth/QQ/login
  - 请求体：`QQLoginDTO`
  - 返回：JWT 字符串

- POST /api/auth/verify-code
  - 请求体：`RegisterDTO`
  - 返回：验证码字符串（示例）

- POST /api/auth/loginOut
  - 请求参数：HttpServletRequest
  - 返回：Result
  - 说明：登出并清理会话/Redis 状态

### 3. 评论（CommentController）
- GET /api/comment/userProfile
  - 返回：当前用户概要信息（UserProfileVO）

- POST /api/comment/publish
  - 请求体：`CommentDTO` { articleId, content, parentId? }
  - 返回：`CommentVO` 或 false（若不可评论）

- GET /api/comment/{articleId}
  - Path 参数：`articleId`
  - 返回：`PageInfo<CommentVO>`（分页评论列表）

---

### 4. 关注（FollowController）
- POST /api/Interaction/follow/{userId}
  - Path 参数：`userId`（要关注/取消关注的用户）
  - 返回：Boolean（true = 当前已关注；false = 已取消）

- GET /api/Interaction/follow/is-following/{userId}
  - 返回：Boolean（是否已关注）

- GET /api/Interaction/follow/followers/count/{userId}
  - 返回：粉丝数（Long）

- GET /api/Interaction/follow/following/count/{userId}
  - 返回：关注数（Long）

---

### 5. 点赞/互动（LoveController）
- POST user/api/Interaction/likes
  - 请求体：`LoveDTO`（包含 entityId、loveType 等）
  - 返回：Result
  - 说明：点赞或取消点赞（由后端决定切换逻辑）

- GET user/api/Interaction/is-liked
  - 请求参数：`entityId`, `loveTypeId`（均为请求参数）
  - 返回：Boolean（是否已点赞）

- GET user/api/Interaction/count
  - 请求参数：`entityId`, `loveTypeId`
  - 返回：点赞数（String）
