# 二次元社区管理后台

一个基于 Vue 3 + Element Plus 的现代化管理后台系统，专为二次元社区网站设计。

## 技术栈

- **前端框架**：Vue 3
- **UI组件库**：Element Plus
- **路由管理**：Vue Router 4
- **状态管理**：(待实现)
- **构建工具**：Vite
- **图表库**：ECharts

## 功能模块

### 1. 数据统计 (Dashboard)
- 用户增长趋势图表
- 内容活跃度统计
- 关键指标卡片展示
- 最近动态列表

### 2. 用户管理
- 用户列表展示
- 搜索和筛选功能
- 封禁/解封用户
- 用户详情查看和编辑
- 批量操作功能

### 3. 内容管理
- 帖子审核管理
- 评论管理
- 内容搜索和筛选
- 批量审核功能

### 4. 动漫库管理
- 动漫列表CRUD操作
- 标签管理
- 搜索和分类筛选
- 动漫信息编辑

### 5. 系统设置
- 角色权限配置
- 敏感词管理
- 批量导入/导出敏感词

## 项目结构

```

│   ├── assets/          # 资源文件

│   ├── views/           # 页面组件

│   │   ├── UserManagement.vue    # 用户管理

│   │   ├── AnimeManagement.vue   # 动漫库管理
```

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 开发模式运行

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 特性

- **响应式设计**：适配不同屏幕尺寸
- **模块化架构**：清晰的代码组织
- **完整的权限管理**：基于角色的权限控制系统
- **数据可视化**：使用ECharts展示统计图表
- **丰富的交互体验**：包含各种表单控件和表格操作
- **模拟数据**：内置测试数据，可直接查看效果

## 注意事项

1. 本项目使用模拟数据，实际部署时需要连接后端API
2. 敏感操作需要二次确认，避免误操作
3. 敏感词过滤功能需要在前端和后端同时实现
4. 建议在生产环境中使用HTTPS协议
